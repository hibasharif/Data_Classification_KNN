# 🌸 DecodeLabs AI — Project 2

### Data Classification using K-Nearest Neighbors (KNN) on the Iris Dataset

An interactive **machine learning web application** that demonstrates how the **K-Nearest Neighbors (KNN)** algorithm works using the classic **Iris dataset**.
It includes a fully visual pipeline for training, evaluation, and real-time prediction with a modern UI.

## 📌 Features

* 📊 Interactive dataset visualization (scatter plot)
* ⚙️ Adjustable KNN parameters (K value, train-test split)
* 🧠 Step-by-step ML training pipeline
* 📈 Real-time evaluation metrics:

  * Accuracy
  * Precision
  * Recall
  * F1 Score
* 🔢 Confusion matrix visualization
* 🌼 Live prediction system using sliders
* 🎨 Modern glassmorphism-inspired UI

---

## 🧠 Machine Learning Workflow

The project demonstrates a full ML pipeline:

1. 📥 Load Iris dataset (150 samples, 4 features, 3 classes)
2. 🔀 Shuffle & split dataset (Train/Test)
3. 📏 Feature scaling using StandardScaler
4. 🤖 Train K-Nearest Neighbors (KNN) model
5. 📊 Evaluate model performance
6. 🔮 Make live predictions

---

## 🎮 How to Use

1. Open the project in browser
2. Go to **Train Model tab**
3. Adjust:

   * K value (neighbors)
   * Train/Test split
4. Click **Run Training Pipeline**
5. View:

   * Metrics
   * Confusion Matrix
6. Go to **Predict tab**
7. Adjust sliders and get real-time classification
